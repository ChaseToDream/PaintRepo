# 画仓 - AI 创作画廊

> 专为 AI 绘画爱好者设计的作品展示平台，支持 ComfyUI 工作流参数记录，纯前端实现，开箱即用

![Version](https://img.shields.io/badge/version-2.0-blue)
![License](https://img.shields.io/badge/license-GPL--3.0-green)

## 特性

### 核心功能
- **瀑布流画廊** - 自适应布局，图片自动排列
- **ComfyUI 参数展示** - 完整记录模型、LoRA、采样器、提示词等生成参数
- **一键复制/下载** - 快速复制全部参数或下载 JSON 文件
- **深浅主题** - 默认赛博朋克暗色风格，支持一键切换

### 技术特点
- **纯前端实现** - 无后端依赖、无数据库
- **Cloudflare R2** - 前端直传直读，全球 CDN 加速
- **开箱即用** - 无需构建工具，无需安装
- **响应式设计** - 完美适配 PC、平板、手机

### ComfyUI 参数支持

| 类别 | 参数 |
|------|------|
| 核心 | 模型、采样器、调度器、步数、CFG Scale、种子 |
| 图像 | 宽度、高度、批次大小、批次数量 |
| 高级 | VAE、CLIP Skip、降噪强度 |
| 扩展 | ControlNet、IP-Adapter、LoRA |
| 提示词 | 正向 Prompt、负向 Prompt |

## 快速开始

### 方式一：使用示例数据测试

1. 下载项目到本地
2. 配置本地模拟 R2（需要部署到云端才能使用完整功能）

### 方式二：完整部署

```bash
# 1. 配置 Cloudflare R2
#    参考 CONFIG.md 完成存储配置

# 2. 部署网站
#    参考 DEPLOY.md 选择部署平台

# 3. 开始使用
#    访问 index.html 查看画廊
#    访问 upload.html 上传作品
```

## 项目结构

```
画仓/
├── index.html          # 首页 - 瀑布流画廊
├── detail.html         # 详情页 - 图片 + ComfyUI 参数
```
