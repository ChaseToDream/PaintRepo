## 详情页

点击首页任意图片，跳转到详情页。

### URL 格式

```
detail.html?name=图片唯一标识
```

### 功能说明

| 功能     | 操作         | 说明                   |
| -------- | ------------ | ---------------------- |
| 查看原图 | 图片展示     | 适配屏幕宽度的高清原图 |
| 查看参数 | 页面下方     | 展示完整 AI 生图参数   |
| 复制参数 | 点击复制图标 | 一键复制单个参数       |
| 返回首页 | 点击按钮     | 返回瀑布流首页         |

### 参数字段

详情页展示以下 AI 生图参数：

| 字段       | 说明            |
| ---------- | --------------- |
| 正向提示词 | Positive Prompt |
| 负向提示词 | Negative Prompt |
| 模型名称   | Model           |
| 采样器     | Sampler         |
| 步数       | Steps           |
| CFG Scale  | 引导强度        |
| 分辨率     | Width × Height  |
| 种子数     | Seed            |
| 批次信息   | Batch 相关      |
| VAE        | 变分自编码器    |

## 上传页

访问 `upload.html` 进入管理员上传页面。

### 访问限制

首次访问需要设置管理员密码。设置后，每次访问需要输入密码验证。

### 功能说明

| 功能     | 操作         | 说明               |
| -------- | ------------ | ------------------ |
| 密码验证 | 输入密码     | 首次设置，之后登录 |
| 选择图片 | 点击上传     | 支持 PNG/JPG/WebP  |
| 填写参数 | 表单输入     | AI 生图参数        |
| 上传     | 点击按钮     | 同步上传图片+参数  |
| 查看配置 | 点击设置图标 | 修改 R2 配置       |

### 上传流程

1. **选择图片**
   - 点击上传区域或拖拽图片
   - 支持格式：PNG、JPG、WebP
   - 文件大小建议不超过 10MB

2. **填写参数**
   - 正向提示词（必填）
   - 负向提示词（可选）
   - 模型名称（必填）
   - 采样器（必填）
   - 其他参数可选

3. **上传**
   - 点击「上传作品」按钮
   - 显示上传进度
   - 上传成功后自动更新 index.json

## 主题切换

画仓支持深色和浅色两种主题模式。

### 操作方式

点击导航栏右侧的太阳/月亮图标切换主题。

### 主题说明

| 主题     | 默认 | 适用场景              |
| -------- | ---- | --------------------- |
| 深色模式 | 是   | AI 绘图展示、深夜使用 |
| 浅色模式 | 否   | 白天、户外环境        |

### 自动记忆

主题选择会保存在浏览器本地存储（localStorage），下次访问自动应用。

## 常见问题

### 图片相关

**Q: 图片显示不出来？**

检查项：

1. R2 公开访问是否配置
2. CORS 是否正确配置
3. 图片 URL 是否正确

**Q: 图片加载很慢？**

可能原因：

1. R2 Bucket 距离用户较远
2. 图片文件过大
3. 网络连接问题

**Q: 支持什么图片格式？**

支持：PNG、JPG、WebP

---

### 上传相关

**Q: 上传失败怎么办？**

检查项：

1. R2 API 密钥是否正确
2. CORS 是否包含 PUT/POST 方法
3. 网络连接是否正常

**Q: 可以批量上传吗？**

目前仅支持单张图片上传。

**Q: 上传后首页不显示？**

上传成功后会自动更新 index.json，刷新首页即可看到新图片。

---

### 配置相关

**Q: 配置丢失了怎么办？**

访问 `upload.html`，重新进行配置。配置存储在浏览器 localStorage 中。

**Q: 如何修改 R2 配置？**

1. 访问 `upload.html`
2. 点击右上角设置图标
3. 修改配置并保存

**Q: 如何修改管理员密码？**

目前版本需要清除浏览器 localStorage 后重新设置。
